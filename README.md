MedShop – Online Medical Store
Overview
MedShop is a full-stack web application developed using Flask that allows users to browse and purchase medicines online. The platform provides a simple and user-friendly interface with secure authentication and efficient data management.

Features
User registration and login system
Browse and search medicines
Add products and manage inventory
Secure authentication system
Responsive and user-friendly UI
CRUD operations for medicines and users

Tech Stack
Backend: Python, Flask
Frontend: HTML, CSS, JavaScript
Database: SQLite (Flask-SQLAlchemy)
Tools: Git, GitHub
